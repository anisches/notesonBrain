# Search And Classification Method

## Search Order

1. Start with the exact concept or claim.
2. Search Google Scholar, Semantic Scholar, PubMed, and journal pages.
3. Use Reddit only for leads, objections, unusual phrasing, and community pointers.
4. Follow citations backward for foundations and forward for newer disputes.
5. Prefer source PDFs, journal pages, DOI records, and author pages.

## Reddit Use

Reddit can be useful for:

- finding layperson objections and confusions
- identifying popular books, videos, and claims
- discovering niche terms that do not show up in academic searches
- finding counterarguments worth checking

Reddit should not be used as evidence unless the object of study is the discussion itself.

## NCBI / PubMed Note

NCBI often hosts abstracts or full-text copies through PubMed and PubMed Central. It is usually not the journal. When recording a source, capture:

- actual journal name
- DOI
- author list
- publication year
- whether it is PubMed abstract only or PubMed Central full text
